Hexo 的 butterfly主题没有自己相应的简历模板，需要一个可以自定义的 html 页面（包括 css 文件和 js 文件）

下面介绍步骤：

新建相关文件夹

在 blog 的源文件夹下找到 source 文件夹，在该文件夹下新建一个文件夹用来储存后继需要部署的 html 文件。该文件夹下可以新建子文件夹，方便存放 css、js文件。
